Post-Processing Effect



Transition styles


This page shows the stlyes I added to the Post-Processing Effects project.

Burning


This style is used in most shows, movies, and video games in order to set up a suspensful feeling like something big is coming.

Screen Melt


This style is inspired by the famous first-person shooting game "Doom". This stlye is usually used when you start the Doom game.

Noise


This stlye of transition I rarely see. It is usally seen in 2D pixelated games. An example of a game that uses this stlye would be Pokemon. This franchise still uses this form of transitions, but it alternates depending on the pokemon.

Swap


This type of transition is usually used in Presentations.

2D Block Dissolve

This type of transition is an interesting one. This style can be used in games shows or even presentations.

Circle Reveal

This stlye was used in 1930's to 1990's cartoons. They use this stlye usually to end the show. An example of one such cartoon that uses this form of transition is Looney Tunes.
